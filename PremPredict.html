Fantasy Premier League Points Predictor

Machine learning-based system that predicts player performance in Fantasy Premier League using historical data and current form.

Technologies Used

Python JavaScript Scikit-Learn React Tailwind

Key Features

  • Weekly point predictions
  • Player form analysis
  • Fixture difficulty ratings
  • Team optimization suggestions

How It Works

This FPL (Fantasy Premier League) predictor script is designed to predict the performance of players for an upcoming gameweek using historical data and machine learning techniques. Here's a breakdown of its functionality:

  • Dataset Handling: Uses player statistics, fixture difficulties, and upcoming fixtures from Kaggle.
  • Fixture Analysis: Identifies upcoming gameweek fixtures and evaluates opponent difficulty.
  • Player Data Preparation: Includes features like cost, form, goals, assists, and fixture difficulty adjustments.
  • Machine Learning Model: A Random Forest Regressor predicts player performance based on attributes and fixtures.
  • Predictions: Outputs a CSV file ranking players by predicted points, considering playing chances and opponent difficulty.
